# 云端法考知识问答项目

这是一个围绕法考资料整理、向量检索和对话式问答搭建的本地项目仓库。

项目目前包含三条主线：

1. 资料清洗
   - 使用 OCR 后处理脚本、法条整理脚本和辅助工具，把原始资料清洗成结构更稳定的 Markdown / TXT。
2. 检索后端
   - 基于 PostgreSQL + pgvector 存储切块和向量，提供本地搜索、健康检查和问答辅助接口。
3. 问答前端
   - 提供面向法考使用场景的对话式前端，支持引用片段展示、模型网关配置、思维导图预览与生成。

## 正式站点

- 当前线上入口：`https://fakaozhixue.cn`
- 当前主站对应的是“法考智学 V2”这一套对话式问答前端
- 仓库里的 [`法考对话前端2.0`](./法考对话前端2.0) 就是这条主链路对应的前端实现

## 使用机制

这个项目当前的实际使用机制是：

1. 用户打开 `fakaozhixue.cn`，进入法考智学 V2 前端。
2. 前端默认使用当前站点内置的检索服务，不需要用户手动填写检索接口。
3. 用户在设置里填写模型 `Base URL`、`API Key` 和模型名。
4. `API Key` 只保存在当前浏览器本地，问答请求由当前服务按 OpenAI 兼容接口代发，前端不直接请求第三方模型域名。
5. 每次提问时，系统会先做站内检索，筛出相关引用片段，再把“问题 + 检索片段 + 对话上下文”一起送给模型。
6. 模型回答返回后，前端会把答案、引用片段、错误提示和思维导图能力一起展示出来。
7. 如果暂时没有填写模型配置，前端仍可以先返回一份基于检索结果整理出的摘要，方便先看资料命中情况。
