# 云端法考知识问答项目

这是一个围绕法考资料整理、向量检索和对话式问答搭建的本地项目仓库。

项目目前包含三条主线：

1. 资料清洗
   - 使用 OCR 后处理脚本、法条整理脚本和辅助工具，把原始资料清洗成结构更稳定的 Markdown / TXT。
2. 检索后端
   - 基于 PostgreSQL + pgvector 存储切块和向量，提供本地搜索、健康检查和问答辅助接口。
3. 问答前端
   - 提供面向法考使用场景的对话式前端，支持引用片段展示、模型网关配置、思维导图预览与生成。

## 仓库里主要有什么

- `法考对话前端2.0`
  - 当前主用前端，基于 `Vite + React + TypeScript`
- `数据库`
  - 搜索服务、向量入库脚本、数据库初始化 SQL、示例环境变量
- `脚本`
  - OCR 清洗、二次清洗、切块、法条整理等脚本
- `整理后文本`
  - 允许公开的清洗后资料原件、法条库文本和来源元数据
- `工具`
  - 本地资料处理辅助工具说明和包装脚本
- `页面`
  - 更早期的检索工作台页面
- `法考对话前端`
  - 旧版前端原型，保留作历史参考，不作为当前主入口

## 当前推荐阅读顺序

1. 先看 `法考对话前端2.0/README.md`
2. 再看 `数据库/README.md`
3. 如需了解资料来源和借鉴说明，再看 `致谢与来源说明.md`

## 当前仓库不包含什么

以下内容明确不上传到这个仓库：

- 云端运行时目录
- 云端导出数据库
- 云端同步压缩包
- 本地运行日志
- 原始 OCR 大文件
- 保密版中间产物

## 适合这个项目的人

- 想做法考知识库问答或本地 RAG 实验的人
- 想把教材、真金题、法条整理成可检索资料的人
- 想在本地把“清洗资料 -> 切块 -> 入库 -> 搜索 -> 前端问答”串起来的人
