# 云端部署说明

这个目录现在只保留云端部署说明和辅助脚本，不保留任何实际云端数据。

## 当前保留内容

- `README.md`
  - 说明仓库里的云端部署边界
- `导出现成本地库.ps1`
  - 本地导出数据库的辅助脚本
- `云端恢复现成库示例.sh`
  - 远端恢复流程示例

## 已明确不入仓的内容

- `runtime/`
  - 运行时打包目录，按需在本地生成
- `exports/`
  - 数据库导出目录，属于云端数据，不允许上传
- `lawqa_runtime_sync*.tar.gz`
- `lawqa_runtime_sync*.zip`

## 当前仓库的云端构建方式

- 使用根目录 `Dockerfile`
- 使用根目录 `docker-compose.cloud.yml`
- 前端构建产物来自 `法考对话前端2.0/dist`
- 数据库初始化脚本来自 `数据库/init`
- 示例环境变量来自 `数据库/.env.pg.cloud.example` 和 `数据库/.env.embedding.cloud.example`

## 使用原则

- 仓库只提交项目代码、脚本、说明和允许公开的清洗资料
- 实际运行包、导出库、临时密钥、线上环境文件一律只留本地，不上传
