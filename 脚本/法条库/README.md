# 法条库脚本目录

本目录用于存放法条库专用脚本。

建议流程：

1. 准备法条清单 `法条库清单.json`
2. 从官方网页/官方接口下载原始文件到 `D:\Onedrive\Obsidian\法考知识点\300_LawText\官方法条下载`
3. 生成项目内正文到 `整理后文本/法条库/`
4. 运行法条切块脚本生成 `切块/法条库/*.jsonl`
5. 用 `数据库/import_chunks_to_pg.py` 导入
6. 用 `数据库/embed_and_update.py --book-name ...` 嵌入
7. 用验证脚本核对书目、chunk 和 embedding

说明：

- 原始官方文件优先保存 `docx`，缺失时保存 `pdf`
- 项目目录只保留可入库正文和来源元数据 sidecar，不再保存官方原件
- 同一法律默认只保留现行有效版本
