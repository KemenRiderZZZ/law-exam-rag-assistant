# 致谢与来源说明

本文档只记录当前仓库里可以明确确认的来源，不把所有依赖包都泛化成“借鉴”。

## 一、明确借鉴或使用到的 GitHub 仓库

### 1. `markmap/markmap`

- GitHub：<https://github.com/markmap/markmap>
- 用途：
  - 本项目在前端中使用 `markmap-lib` 和 `markmap-view` 渲染法考知识点思维导图。
- 对应目录：
  - `法考对话前端2.0/src/components/OfficialMarkmapView.tsx`
  - `法考对话前端2.0/src/lib/mindmapPreview.ts`
- 说明：
  - 这里主要是基于其现成能力做集成、兼容处理和移动端预览适配，不是直接整仓复制源码。

### 2. `pgvector/pgvector`

- GitHub：<https://github.com/pgvector/pgvector>
- 用途：
  - 本项目的本地向量检索部分建立在 PostgreSQL 的 `pgvector` 扩展之上。
- 对应目录：
  - `数据库/init/01-init-pgvector.sql`
  - `数据库/README.md`
  - `docker-compose.yml`
  - `docker-compose.cloud.yml`
- 说明：
  - 这里主要借用了 `pgvector` 的数据库扩展能力、安装流程和配套使用方式。

### 3. `opendatalab/MinerU`

- GitHub：<https://github.com/opendatalab/MinerU>
- 用途：
  - 本项目在 OCR / 文档转 Markdown 工作流中使用 MinerU 作为资料解析工具之一。
- 对应目录：
  - `工具/MinerU`
- 说明：
  - 仓库里保留的是本地调用说明、包装脚本和使用记录，不是 MinerU 项目的完整源码镜像。

## 二、非 GitHub 仓库但确实影响过当前项目结构的来源

### Google AI Studio 导出模板

- 影响范围：
  - `法考对话前端`
  - `法考对话前端2.0` 的少量历史模板痕迹
- 说明：
  - 仓库中的旧版前端最早来自 Google AI Studio 导出的前端原型，后续在本地逐步改造成适合法考问答的结构。
  - 目前 `法考对话前端` 被保留为历史参考目录，当前主用前端是 `法考对话前端2.0`。

## 三、本仓库自己的增量工作

除了上面的开源基础能力和历史模板来源，这个仓库里真正属于本项目自身持续演进的部分主要包括：

- 法考资料的 OCR 清洗与二次清洗脚本
- 法条库整理脚本与来源元数据组织
- 面向法考场景的检索接口封装
- 对话式问答前端
- 模型网关测试、设置持久化和错误提示流程
- 思维导图生成、兼容预览和节点展开体验
